# Brief Director — Fabric Showcase Pipeline

## 何时使用

当用户需要为面料/纺织品制作展示广告时使用本技能。你负责收集面料真理（fabric_facts）和广告意图（ad_intent），引导用户完成旁白稿和场景大纲，产出唯一工件 `fabric_brief`。

**面料是真理之源。** 其他一切——灯光、构图、动效、BGM——都是为了帮助观众理解这块面料的质感、成分和用途。不得虚构面料属性。

## Prerequisites

| Layer | Resource | Purpose |
|-------|----------|---------|
| Schema | `schemas/artifacts/fabric_brief.schema.json` | Artifact validation |
| Prior artifacts | 用户提供的面料参考图和面料事实 | Fabric truth source |
| Tools | 无（brief 是纯信息收集阶段，不调用生成工具） | Human interview only |
| Playbook | custom_only | No fixed playbook |

## 生产原则

将每一个属性区分为：

- **文档化事实（fabric_facts）**：成分比例、手感、克重、用途、参考图、色彩、季节。这些由用户提供，不可虚构。
- **创意选择（ad_intent）**：平台、时长、比例、风格导向、场景结构。这些可以协商调整。
- **生产启发式（production heuristics）**：质感展示角度、光线对面料的影响、动态展示方式、prompt 修复策略、QA 检查项。

**创意选择不可覆盖文档化事实。**

## 面料真理表（Fabric Facts）

在开始生成任何视觉素材前，收集或推断以下信息。如果信息缺失，对于低风险创意选择可做可见假设，对于高风险事实必须暂停询问用户。

### 必填字段

| 字段 | 语义 | 示例 |
|------|------|------|
| `composition` | 成分比例（不可虚构） | `55% linen, 45% cotton` |
| `texture` | 触感/质感描述 | `亚麻筋骨，棉柔软，亲肤透气` |
| `applicable_products` | 用途/适用产品列表 | `衬衫, 连衣裙, 帆布包` |
| `reference_image` | 原始面料参考图路径 | `assets/fabric-original.jpg` |
| `color_palette` | 面料色彩描述 | `复古红格（砖红+深棕），哑光，中厚` |

### 可选字段

| 字段 | 语义 | 示例 |
|------|------|------|
| `season` | 适用季节 | `三季皆宜` |
| `weight` | 面料克重 | `中厚，约 180g/m²` |
| `drape` | 垂坠性 | `中等垂坠，有筋骨` |
| `stretch` | 弹性 | `无弹` |
| `sheerness` | 透明度 | `不透明` |
| `finishing` | 后整理工艺 | `水洗复古做旧` |
| `care` | 养护说明 | `冷水手洗，低温熨烫` |
| `origin` | 产地（如有商业声明） | `中国织造` |

### 真理表不完整时的处理

- 参考图清晰 → 可从图中推断色彩、纹理、透明度，但**成分比例必须由用户确认**
- 参考图模糊或缺失 → 避免精确的质感细节描述，避免后续生成中被 AI 幻觉填充
- 成分/用途未知 → **暂停**，要求用户提供，不可猜测

## 风险评估

如果面料广告涉及以下任一情况，在 brief 阶段标记并在 review_focus 中强调：

- **成分声明**：如"100% organic cotton"、"recycled linen"等受监管的纤维/环保声明
- **产地声明**：如"Made in Italy"、"中国织造"等可能受标签法约束的声明
- **性能声明**：如"抗菌"、"防UV"、"透气速干"等需要 substantiation 的声明
- **真人模特**：如需使用真人/网红/员工的肖像
- **合成模特**：如使用 AI 生成的人物形象，需检查平台 AI 披露要求

**如果用户提供了高风险声明但无法提供证据**，将声明降级为主观描述：
- "抗菌" → "亲肤透气"
- "sustainable" → "天然混纺"
- "Made in Italy" → 删除产地信息

## 广告意图（Ad Intent）

收集用户对广告产物的要求：

| 字段 | 语义 | 可选值 |
|------|------|--------|
| `platform` | 目标平台 | `xiaohongshu` / `douyin` / `bilibili` |
| `duration` | 目标时长（秒） | 通常 15-20 |
| `aspect_ratio` | 画面比例 | `9:16`（小红书/抖音）/ `16:9`（B站）/ `1:1` |
| `style_direction` | 风格导向 | 如"复古、文艺、质感"、"现代、简约、高级" |
| `brand_name` | 品牌名（如有） | 如"铃月布行" |

## 场景语法（Shot Grammar）

面料广告使用以下镜头类型来平衡质感展示和产品信息：

| 镜头类型 | 目的 | 时长参考 |
|----------|------|----------|
| **面料特写** | 展示纹理、色彩、编织细节 | 3-5s |
| **面料飘动** | 展示垂坠性、重量、流动感 | 3-4s |
| **手部触碰** | 展示触感、柔软度、筋骨 | 3-4s |
| **模特上身** | 展示面料在服装上的效果 | 3-5s |
| **产品线画（可选）** | SVG 线稿展示用途产品 | 2-3s |

### 典型 4 幕结构（15-17 秒）

```
0-5s   面料特写（纹理+色彩+标题）
5-9s   面料飘动（垂坠+重量）
9-13s  手部触碰（触感+柔软度）
13-17s 模特上身（穿着效果+收尾）
```

### 视频节拍写法

不要写泛 mood，写具体 beat：

```
0.0-5.0s: 面料平铺特写，灯光从左滑入揭示纹理，标题"复古红格"浮现
5.0-9.0s: 面料被风轻吹，展示垂坠和重量，红格纹在光线下变化
9.0-13.0s: 手轻轻抚过面料表面，展示触感和柔软度
13.0-17.0s: 模特背面穿着连衣裙，Ken Burns 慢推，收尾
```

## 旁白稿（Voiceover Script）

旁白稿必须忠于 fabric_facts，不可夸大宣传。

### 写作规则

- 品牌名 + 面料名 + 成分复述 + 触感描述 + 适用产品
- 20-40 字（对应 13-20 秒朗读量）
- 不使用未经证实的高风险声明
- 不含个人属性暗示（如"改善你的肤感"）

### 示例

```
铃月布行，复古红格。五十五麻四十五棉，天然混纺，亲肤透气。
亚麻的筋骨，棉的柔软，适合做衬衫、连衣裙、帆布包。三季皆宜，设计师臻选。
```

### 声明降级检查

| 旁白原文 | 风险 | 降级后 |
|----------|------|--------|
| "有机棉，零碳生产" | 未经证实的环保声明 | "天然混纺，亲肤透气" |
| "抗菌防臭" | 性能声明需证据 | 删除，或"清新舒适" |
| "意大利进口" | 产地声明需证据 | 删除产地信息 |

## Prompt 脚手架

后续 assets 阶段生成图片和视频时，brief 阶段需要准备好以下 prompt 结构要素：

### 面料图片 Prompt 结构

1. **生产意图**：面料特写 / 飘动 / 手触碰
2. **面料真理锁**：精确的色彩、纹理、成分表现、不可改变的细节
3. **光线/相机**：光线方向、微距/特写、景深、构图
4. **品牌调性**：复古/文艺/简约/高级
5. **负面约束**：不发明纹理、不改变色彩、不添加不存在的光泽
6. **交付要求**：比例、安全区、无文字（除非已批准）

### 示例：面料特写 Prompt 锚定

```text
面料特写产品图，复古红格面料平铺。
真理锁：砖红色与深棕色交织的格子图案，哑光表面，可见亚麻竹节纹理，
中厚面料，不透明。不可改变格纹颜色、比例、纹理质感。
光线：柔和侧光从左上方 45 度入射，展示面料立体感和纹理起伏。
相机：微距特写，景深浅，对焦在格纹交叉点。
风格：复古文艺，质感优先。
比例：竖版 9:16，上半部留白用于标题。
不要添加光泽、不要改变红格颜色、不要发明新的纹理、不要文字。
```

## 交付变体规划

在 brief 阶段为每个目标平台规划变体：

| 平台 | 视频比例 | 封面比例 | 时长 |
|------|----------|----------|------|
| 小红书 | 9:16 竖版 | 3:4 (1080×1440) | 15-17s |
| 抖音 | 9:16 竖版 | 9:16 (1080×1920) | 15-17s |
| B站 | 9:16 或 16:9 | 16:9 (1080×608) | 15-60s |

每个变体需保留：
- 面料可见性（特写镜头在所有比例下都能看清纹理）
- 文字安全区（标题不超出平台 UI 覆盖区）
- 旁白和 BGM 一致性

## 面料 QA 检查清单

在 brief 阶段产出前，自查以下项目：

### 面料真理
- [ ] composition 完整且由用户确认，非 AI 推断
- [ ] texture 描述具体（不是泛泛"柔软""舒适"）
- [ ] applicable_products 与面料特性匹配
- [ ] reference_image 存在且清晰
- [ ] color_palette 精确（含光泽/哑光描述）

### 声明安全
- [ ] voiceover_script 中无未经证实的高风险声明
- [ ] 成分/产地/性能声明有证据或有降级
- [ ] 无个人属性暗示或身体羞辱语言

### 场景可行性
- [ ] 场景数量合理（3-5 幕）
- [ ] 每幕时长分配合理，总和匹配 duration
- [ ] 每幕的 assets_needed 可由现有工具链生产
- [ ] 目标平台的画面比例已确认

### 交付规划
- [ ] 至少一个平台的变体已规划
- [ ] 封面风格方向已确定（字体、调性）
- [ ] 品牌名是否需要出现在画面中已明确

## 常见错误

- **跳过参考图直接写 brief**：没有参考图就无法锁定面料真理，后续生成会被 AI 幻觉填充
- **成分比例凭空猜**："看起来像棉" → 必须问用户，不能猜
- **旁白含未经证实声明**：如"有机""抗菌""进口"等需要证据的词汇
- **场景过多**：15 秒做成 8 幕，每幕不到 2 秒，节奏碎裂
- **忽略封面规划**：封面的字体/调性和视频风格不一致
